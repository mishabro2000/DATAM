# DATAM 

Практическое пособие по старту в области инженерии данных (Data Engineering), Data Science и ML

## О проекте
DATAM — это интерактивный путеводитель, который объясняет, как устроена работа с данными, что именно нужно учить новичку и как запустить свои первые pet-проекты

**Текущий статус:** Разработка (Готово на ~15%)
**Планируемый релиз:** Начало / середина августа 2026 года

##  Кому это подойдет?
- **Data Scientists & Data Engineers** (уровня стажер/джуниор), желающим структурировать знания
- **ML-разработчикам**, которым нужно подтянуть навыки работы с пайплайнами данных
- **Новичкам**, которые хотят понять, чем занимаются инженеры данных и с чего начать

## Кому это НЕ подойдет?
- Разработчикам из других сфер (веб, мобилки, геймдев), не связанных с Big Data
- Тем, кому абсолютно не интересна тема обработки, хранения и анализа данных

## Что внутри (в разработке)?
- [ ] Статьи из Habr которые стоит будет прочитать
- [ ] Дорожная карта (Roadmap) технологий Python и SQL(sqlite)
- [ ] Готовые шаблоны и разборы базовых программ для DE
- [ ] Идеи и гайды по созданию первых pet-проектов

##  Мой стек в этом проекте
Мой стек достаточно простой тут нету Spark или Docker это пока что для новичков будет сложно
В основе здесь будет только два инструмета:
- **Python** (чистый код, автоматизация, обработка данных)
- **SQL / SQLite** (проектирование реляционных баз данных и запросы)
